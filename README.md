Visual login screen design in Android Studio.

Includes the screens main, welcome, login and sign up.
Buttons navegate between screens 

Validation of inputs: 
    
		- Validate name accepting only letters 
    
		- Validate email 
    
		- Validate phone number having 10 digits and accepting only numbers
    
		- Validate passwords beeing the same
    
		- Unable button if fields are empty or wrong

Implemented using Column, Row, Button, Text, Image, OutlinedTextField, Navegation and Modifier.
